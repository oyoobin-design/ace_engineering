## 1) 시장분석 — ESS

**도구**: Market 탭의 시장 규모·성장 차트
**현재**: EIA로 미국 배터리 설치용량 실시간 수집(🟢), `ess-market.ts`에 글로벌·지역별 용량이 정적으로 있음. 단 글로벌 수치는 "재구성/전망"이라 신뢰도 라벨이 붙어있는 상태.
**추가 필요**: ① 신뢰할 수 있는 글로벌·지역별 실측 데이터 ② "누가 어디로 ESS를 수출입하는가"의 교역 흐름(에이스가 수출기업이므로 핵심).

**투입 API**

- **UN Comtrade API** — 글로벌 배터리/컨테이너 교역 흐름
  - 역할: HS코드 **850760(리튬이온전지)**, **8609(컨테이너)** 기준으로 국가 간 수출입액·물량을 가져와 "ESS 수요가 실제로 어느 나라로 흐르는지"를 시각화.
  - 무료: 회원가입 후 키 발급, **호출당 10만 건 / 일 500회**까지 무료. ~200개국, 세계 교역의 99% 커버.
- **관세청 수출입무역통계 (data.go.kr)** — 한국의 ESS 관련 수출 목적지
  - 역할: 품목별·국가별 수출입실적(HS 2~10단위)으로 *한국발 ESS·배터리·컨테이너 수출이 어느 나라로 늘고 있는지* 추적 → 에이스 같은 수출기업 관점의 시장 신호.
  - 무료: 공공데이터포털 활용신청, 개발계정 트래픽 10,000.

## 1) 시장분석 — Data Center (JD가 명시한 신규시장)

**도구**: 신규 **Data Center 탭** (JD 우선순위상 ESS 다음)
**현재**: `ess-market.ts`의 `dcMarket`에 "100MW DC당 ESS 20~40MWh 필요 = 에이스 신규시장" 메모와 연도별 MW 추정만 있음. **라이브 소스 없음(🔴).**
**추가 필요**: 데이터센터 전력수요·증설 → ESS 수요로 환산하는 라이브 신호.

**투입 API (솔직한 한계 포함)**

- ⚠️ **데이터센터 시장은 쓸만한 무료 라이브 API가 사실상 없음.** 좋은 데이터는 JLL·CBRE·Synergy·Data Center Map 등 **유료 리포트**임.
- 현실적 대안 3가지:
  1. **산업 리포트 수치를 정적 데이터로 내장** (JLL/Technavio 등 공개 헤드라인 수치 — 출처·날짜 명기). 분기 1회 수기 갱신.
  2. **하이퍼스케일러 CapEx**를 SEC EDGAR/FMP로 추적 (Microsoft·Amazon·Google 등 — DC 투자 선행지표). → DC 붐의 객관 신호.
  3. **뉴스 RSS**(이미 구축됨)에서 "data center / AI power / BESS" 키워드로 증설·전력계약 이벤트 포착.
- ✅ 정직한 포지셔닝: "DC는 실측 API가 없어 **공개 리포트 + 하이퍼스케일러 투자 + 뉴스 신호를 조합**해 추정한다"고 명시하면 오히려 데이터 한계를 아는 기획자로 보임.

## 1) 시장분석 — 신규시장 (일본·스페인·중동 등)

**도구**: Market 탭 내 국가 비교 뷰
**현재**: 거의 없음(🔴).
**추가 필요**: 국가별 ESS 잠재력(전력수요·재생에너지 비중·정책)을 비교해 "다음에 어느 시장을 노릴지" 우선순위화.

**투입 API**

- **World Bank Indicators API** — 국가별 거시·에너지 지표
  - 역할: 국가별 **전력생산·재생에너지 비중(EG.ELC.RNEW.ZS 등)·전력소비**를 가져와 시장 매력도 비교. ESS 수요는 재생에너지 변동성과 직결되므로 신규시장 스크리닝에 적합.
  - 무료: **키 불필요**, JSON 지원. 가장 붙이기 쉬움.
- (보조) **관세청 / UN Comtrade** — 해당 국가의 배터리·ESS 교역 증가세로 수요 확인.

## 2) 사업계획 수립·관리 — 수요예측

**도구**: Forecast 탭 (파트너 수주 → 에이스 발주량 역산, 컨테이너 1개 ≈ 3~5MWh)
**현재**: Fluence·Powin 분기 수주가 **하드코딩된 정적 데이터(🔴)**. 로직은 좋으나 데이터가 가짜.
**추가 필요**: 공개 파트너사의 **실제 수주 공시·보도자료를 자동 수집** → 역산에 투입.

**투입 API**

- **SEC EDGAR Full-Text Search API** — 미국 상장 파트너의 수주 보도자료
  - 역할: 8-K의 **EX-99.1(보도자료)** 전문 검색으로 "award / GWh / BESS / project" 키워드 수주 발표를 자동 포착 → MWh 추출 → 발주량 역산.
  - 무료: **키 불필요**, 2001년 이후 전 공시 전문 검색.
- **DART OpenAPI** — 국내 파트너·SI의 공시
  - 역할: 국내 상장 파트너(삼성SDI·LG에너지솔루션·SK on 등)의 주요사항보고·수주 공시 수집.
  - 무료: 키 발급.

## 2-b) 리스크관리 — 파트너 건전성 (제2의 Powin 조기경보)

**도구**: 파트너 모니터링 (현재 Customer Intelligence 탭을 다수 파트너로 확장)
**현재**: SEC EDGAR로 **Fluence 1곳만** 백로그·매출 추적(🟡).
**추가 필요**: 에이스가 홈페이지에 **공개한 파트너사**들의 재무 건전성을 한 화면에서 모니터링 → "또 다른 Powin"이 될 곳을 조기 포착. (집중 리스크 관리 = 정통 사업기획)

**투입 API (파트너 국적별로 분담)**

- **SEC EDGAR companyfacts** (이미 사용 중) — 미국 상장 파트너
  - 대상 예: Fluence, Bloom Energy, Caterpillar, Honeywell, GE, Tesla 등. 매출·백로그·부채 추적.
- **DART OpenAPI** — 국내 상장 파트너
  - 대상 예: 삼성SDI, LG에너지솔루션, SK on(SK이노), 두산, 효성중공업, LS일렉트릭. 재무제표·주요사항.
- **FMP (Financial Modeling Prep)** — 위 두 곳이 못 잡는 글로벌 기업 보조
  - 역할: ABB·Siemens·Sumitomo 등 비미국·비한국 상장사 재무를 한 API로.
  - 무료: **하루 250회 / 미국 5년치 / 30일 500MB** — 한도 빡빡, 포트폴리오 평가용으로만. 핵심 의존 X.

## 2-c) 경쟁사 동향

**도구**: Competitor 탭
**현재**: 정적 경쟁사 카드 + 구글뉴스 RSS 이벤트 자동분류(🟢).
**추가 필요**: 경쟁사 **재무·증설 공시**(뉴스보다 신뢰도 높음).
**투입 API**: **DART**(신성에스티 등 국내 추격사 — 켄터키 공장 증설 같은 공시), **SEC EDGAR**(해외 상장 경쟁사).
**보정 포인트**: CATL은 에이스 홈페이지에 *파트너로도* 올라가 있음 → 배터리는 경쟁이지만 외함은 고객일 수 있는 **coopetition(협력+경쟁)**. 경쟁사 모델에 이 양면성을 반영.

## 3) 업무 Process 개선

**도구**: 자동화 그 자체가 증거물.
**현재**: 뉴스 3단계 AI 필터 + 인메모리 캐시 + PPT 자동생성 = 이미 "수작업 → 자동화" 사례(🟡, 단 수동 새로고침).
**추가 필요**: **스케줄러** — 매일 아침 데이터 자동 갱신 + 브리프 자동 발송. "사람이 버튼 누르던 걸 시간표대로 무인화"가 process 개선의 완성.
**투입 API**: 외부 데이터 API 아님. (cron / 스케줄 작업 + 메일·메신저 발송)

## 4) 전사 주요 업무 진행상황 보고

**도구**: Report Studio
**현재**: 뉴스+EIA → GPT-4o → **.pptx 자동 다운로드**(🟢). **PPT 우대요건을 정확히 충족.**
**추가 필요**: ① 위에서 추가한 데이터(파트너 건전성·수요예측·DC·신규시장)를 보고서 슬라이드에 반영 ② **엑셀(.xlsx) 출력 옵션** — 자격요건에 PPT와 함께 **엑셀**이 명시돼 있으므로, "원클릭 엑셀 데이터팩" 기능을 더하면 우대요건 2개를 동시에 증명.
**투입 API**: 신규 외부 API 없음. 기존 데이터 통합 + xlsx 생성 라이브러리.

## 권장 구현 우선순위

1. **DART + SEC 풀텍스트** → 파트너 건전성 모니터링 + 수요예측을 *진짜 데이터*로 전환. (가장 사업기획다움, Powin 조기경보 서사 완성)
2. **World Bank** → 신규시장 국가비교. (키 불필요라 즉시 가능, 비용 0)
3. **관세청 / UN Comtrade** → 수출 흐름 시장분석. (에이스=수출기업이라 설득력 큼)
4. **엑셀(.xlsx) 출력** → PPT에 더해 엑셀 우대요건 동시 충족.
5. **스케줄 자동화** → Process 개선 증거.
6. **데이터센터** → 무료 라이브가 없으므로 *정적 리포트 + 하이퍼스케일러 CapEx + 뉴스*로 정직하게 조합. 욕심내지 말 것.
